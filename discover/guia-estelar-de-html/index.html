Exercícios de HTML

Neste exercício irei escrever dois parágrafos dando ênfase em algumas palavras e inserindo um link de "saiba mais". Aqui, destaco a palavra "HTML".

O segundo parágrafo contém o link de acesso para o site do Google*. Para saber mais, clique aqui e você será direcionado para a nova página.
*Aqui está contida a tag abbr que trata sobre abreviação.

Atenção: Acesse o código fonte para verificar as tags e estruturação da página.

Citações

O Elemento HTML <blockquote> (ou HTML Block Quotation Element) indica que um texto externo foi citado.

Representação de códigos

Para representar códigos usamos as tags:
<code></code> — muda a cor do conteúdo para representar código.
<pre></pre> — tag para criar um bloco de código, mantém os espaços em branco.
